# GVR walk kit
<br>?
<br>Now, your 3 tatami mats will become an open world that will spread all the way. Installation is unnecessary, you can easily create a GVR operating environment with a controller using the web
<br>?
<br>## easy explanation
<br>?
<br>You can walk in the VR space using the remaining smartphone as a controller.
<br>First, connect the smartphone that launches the GVR walk kit and the smartphone that serves as the controller to the same network.
<br>When starting up, the IP address is displayed, so if you enter it in the browser of the smartphone that uses it as a controller,
<br>Attach to the center. You can go forward or sideways by reading the round trip of the foot.
<br>

<br>## 簡単な説明
<br> 
<br>余っているスマホをコントローラーにしてVR空間を歩けます。
<br>まず、GVR walk kitを起動するスマホとコントローラにするスマホを同じネットワークに繋ぎます。
<br>起動するとIPアドレスが表示されるので、それをコントローラにするスマホのブラウザに入力し繋がったら足のもも
<br>にあたりに装着します。足の往復を読みとり前に進んだり横に進んだりすることができます。
